# Musify

Android music streaming app built with Kotlin and Jetpack Compose.

## Tech

- Kotlin / Jetpack Compose
- MVVM + Clean Architecture
- Hilt, Retrofit, Room, Coil

## Setup

Open in Android Studio, sync Gradle, and run.

Configure your backend URL in `local.properties`:
```
api.base.url=http://your-backend:8080
```

## Backend

Requires [Musify-Backend](https://github.com/Rostanic20/Musify-Backend).
